# Logos de la Marca

Coloca aquí los archivos de los logos de la marca:

- `logo-icon.png` o `logo-icon.svg` - El icono del speech bubble con rayo rosa (usado en Navbar y Footer)
- `logo-full.png` o `logo-full.svg` - El logo completo con texto (opcional, para usar en otros lugares)

## Formatos recomendados:
- PNG para mejor compatibilidad
- SVG para mejor calidad y escalabilidad
- Tamaño sugerido para el icono: 40x40px a 80x80px
- Tamaño sugerido para el logo completo: Ancho máximo de 200px

## Notas:
- Las imágenes se cargan desde `/images/` (carpeta public)
- El código tiene fallback al texto si las imágenes no están disponibles
